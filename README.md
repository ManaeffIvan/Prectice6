# Задача 1

## Условие задачи

### Вариант 13

> Дан массив из N элементов. Вводится число K – количество блоков (K<<N). Упорядочить элементы по убыванию в каждом блоке.

### Наборы тестов с результатами работы программы

1. Генерация тестов состоит из дву пунктов:
    - Ручная через ввод
    - Автономная через 4 теста
